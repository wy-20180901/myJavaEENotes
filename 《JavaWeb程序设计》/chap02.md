# HTML基础

## 静态网页制作

- 常用标签及属性

  ```html
  <title></title>
  <hn>
  	n从1到7
  </hn>
  <p>
  	常用属性有 align
  </p>
  <hr> 属性有size, width, align, color, noshade
  <font>
  	常用属性有size, face, color
  </font>
  <b>设置为粗体</b>
  <u>下划线</u>
  <i>设置为斜体</i>
  <sup>设置为上标</sup>
  <sub>设置为下标</sub>
  <ul>
      <li>无序列表</li>
      <li>1</li>
      <li>1</li>
  </ul>
  <ol>
      <li>有序列表</li>
      <li>2</li>
      <li>3</li>
  </ol>
  <table>
      <caption>表格标题</caption>
      <tr>
      	<td>1.1</td>
          <td>1.2</td>
      </tr>
      <tr>
      	<td>2.1</td>
          <td>2.2</td>
      </tr>
  </table>
  <!-- 属性有align, bgcolor, border, width, height, bordercolor, cellpadding, cellspacing, rowspan, colspan -->
  <a href="https://github.com/KingdeGuo/myJavaEENotes"></a>
  <img src = "imgpath" width, height, border, align, alt>
  <form action="submit place">
      <input type=text,radio,checkbox,reset,button,submit,image>
      <texearea></texearea>
      <select>
          <option selected></option>
          <option></option>
      </select>
  </form>
  <frameset cols="30% 70%">
      <frame src="a.html" norsize scrolling="no" name="left"></frame>
  	<frame src="b.html" norsize scrolling="no" name="right"></frame>
  </frameset>
  ```
